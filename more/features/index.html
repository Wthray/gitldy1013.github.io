# 项目描述

北航天华针对在线云教育方向的项目主要功能 : 基础信息管理，学生选课，排班管理 学生成绩管理 老师管理 课表管理 考务管理等。

# 职责描述

# 北苑北交通枢纽信息化管理平台（2018/05-2019/05）

# 项目描述

在北苑北综合交通枢纽应用软件平台基础上，为枢纽公司级综合运行监测与应急指挥平台做好技术储备及接口设计，实现对全市多个客运枢纽的综合运行监测、异常报警与联动处置及信息联动诱导发布，保障全市综合交通枢纽安全稳定运营，为行业政策规划提供决策支持

# 职责描述

针对项目背景梳理项目开发流程，对接项目需求完成页面后台基础搭建，启动项目开发，组件项目团队，研发工作，项目中应用的技术有websocket推送消息到web页面，视屏流后台转码处理服务，单点登录方案解决，单用户登录和权限模块设计等。

# CDN后台管理项目 (2017/03-2018/05)

# 项目描述

移动后台流量管理平台，针对移动集团提出的需求，针对zabbix服务器监控获取修改端口和设备提供的内部管理系统。主要应用技术框架springboot,docker容器,复杂表格校验的poi文件导入导出,redis的session共享和远程调用服务(dubbo)技术实现。

# 责任描述

本人负责对接移动集团需求变动，项目设计分析，后台架构搭建 项目代码编写 逻辑实现 部署服务器 搭建环境等工作。按照公司提供的BBoss文档开发一些发起和处理请求的接口(RESTFUL风格)。项目的打包，数据库数据录入迁移，jenkins搭建等。同时也会有前期项目的一些维护交接工作。

# 和创空间后台管理项目 (2016/03-2017/03)

# 项目描述

服务中国移动双创活动，为双创业务平台和系列产品（和创空间网站及APP）提供相应的内容管理、审批管理、数据管理等内容。项目模块较多，可在负责模块中查看具体项目功能。中国移动和创空间官网

# 责任描述

参与项目初始架构搭建，数据库表设计，项目核心模块编码，主要包括：系统用户，角色，权限管理(主要应用拦截器实现)，日志管理 ，短信群发(多线程满足一定程度的高并发)，自定义查询，普通用户的业务数据管理，web内容发布资讯管理，用户管理，群发列表，项目审批自发创意审批，评审管理涉及部分后台编码和模块的前后台联调开发，findbugs，checkstyle，codereview，代码重构，服务容器配置，redis配置自动化部署以及移动总部上级安排的一些编写开发文档等工作。

# 移动内部物联网云平台（2015/08-2016/02）

# 项目描述

为了促进物联网业务的发展，增加用户粘度，方便用户使用，杭研建设M2M业务网关系统。现阶段首先与M2M卡管平台和杭研的写卡平台（以下简称写卡平台）对接，向用户提供M2M卡的状态查询，短信收发，号码切换等服务；未来，该系统可集成更多的业务能力，向用户提供更多，更便捷的服务。项目采用SSM的架构模式，采用注解试开发，maven管理jar包，svn的版本控制方式，项目功能是为移动提供各种业务类型的短信模板，邮箱邮件模板统一云管理的功能。

# 责任描述

我主要负责为为写卡平台提供服务接口，应用移动云平台服务接口以及后台权限管理和日志管理模块。通过从webService获取数据获得数据提供Http+json的方式将数据返回给写卡平台，编写了短信模块，模板模块的相关列表修改,删除操作的编码任务，findbugs,checkstyle。整体来说项目进度不是很忙，所以期间做了几次技术点分享：类似docker容器应用，mysql索引的应用，代码书写规范等。

# 贯众健走后台管理系统 （2015/05-2016/08）

# 项目描述

贯众健走，是一款企业级运动健身服务平台。该项目是移动推出的后台管理平台。其中包括后台管理人员的登录，用户管理，组织机构管理，活动管理，计步统计管理，app更新管理，反馈意见管理模块。采用的是springMVC+Spring+Mybatis后台框架。项目中还应用到了redis解决session共享，定时器控制后台数据库定时批量更新计步统计数据，dtree树形结构，bootstrap等其他技术。

# 责任描述

我在这个项目中负责了组织机构管理模块：其中包括部门管理和员工管理。部门树形结构的浏览和的基本的数据库操作。员工的数据库增改，按照字段条件查询操作，以及excel文件的导入导出操作。计步统计管理模块，分为月管理和日管理。分页+条件查询+excel表格文件的导出。app更新管理模块，新版本的apk文件上传以及版本信息在数据库中的更新，列表的条件查询。以及活动管理及其他模块的excel文件导入导出功能。

# 旺佳商品综合管理系统 （2014/08-2015/03）

# 项目描述

旺佳连锁超市有限公司是四川省内江市内一家私营公司，我公司为旺佳乐连锁超市开发了该B/S架构，超市后台管理系统。用于建立网上终端、营销案和号卡在线销售及相关辅助功能，系统后台主要功能包括商品管理、类目管理、营销案管理、订单管理、供货商管理、配送商管理、会员管理，仓储管理、对账管理、互动管理、权限管理；系统前台主要功能模块包括商品类型、商品检索、首页-频道页-单品页、营销专题、订单支付、购物流程、客户中心、帮助中心；对提高客户服务效率和客户满意度能够起到较好的作用。

# 责任描述

商品管理模块：其中包括品牌管理，商品录入/上下架管理，商品添加，商品库存管理，静态页面发布商品前台首页：其中主要负责首页商品列表检索，首页上动态展示筛选条件，商品详情页静态化以减轻系统压力，个人中心，包括用户的登录，个人信息的管理，收货地址的管理，用户所下的订单的管理，购物车的管理。

标准的ERP项目，我主要负产品信息管理、仓库信息管理、库存管理、库存预警管理、商品损坏记录管理、进退货管理、供应商管理。
